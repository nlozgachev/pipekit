## Imports

The library is split into three entry points. Import only what you need:

```ts
import { Option, Result, Task } from "pipelined/core";
import { pipe, flow } from "pipelined/composition";
import { Brand } from "pipelined/types";
```

| Entry point             | Contains                                                                                                                                                  |
| ----------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `pipelined/core`        | `Option`, `Result`, `Validation`, `Task`, `TaskResult`, `TaskOption`, `TaskValidation`, `These`, `RemoteData`, `Lens`, `Optional`, `Reader`, `Arr`, `Rec` |
| `pipelined/composition` | `pipe`, `flow`, `compose`, `curry`, `tap`, `memoize`, and other function utilities                                                                        |
| `pipelined/types`       | `Brand`, `NonEmptyList`                                                                                                                                   |

## Requirements

- **Node.js** 22 or later (for npm/pnpm/yarn/bun installs)
- **Deno** 2 or later
- **TypeScript** 5.4 or later with `strict` mode enabled

Strict mode is required for the type-level guarantees — particularly exhaustiveness checking in `match` branches and narrowing in type guards — to work correctly.
